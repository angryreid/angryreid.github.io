简话Virtual DOM

随着 React 的兴起，Virtual DOM 的原理和实现也开始出现在各大厂面试和社区的文章中。其实这种做法早在 d3.js 中就有实现，是 react 生态的快速建立让它正式进入了广大开发者的视角。

Virtual DOM

1. 定义

是用JS语言来描述页面DOM结构的一个对象。

demo

2. 用途

在React/Vue这类的MVVM框架render为页面的过程中，将数据与模板结合起来构建为虚拟DOM对象
然后将虚拟DOM映射为真正的页面DOM元素。

3. 意义

虚拟DOM大大的提高了页面的性能
通常在我们的MVVM框架编写的页面中，数据发生了改变，页面也会同步的更新
就是新的虚拟DOM与旧的虚拟DOM之间通过DIFF（差分算法）进行比较，
定位出两者发生改变的部分，然后针对这一部分变化的DOM与真正的DOM进行替换操作。
